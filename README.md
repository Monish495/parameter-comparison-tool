# Parameter Comparison Tool

A Java-based full-stack web application designed to compare datasets and identify structural and value differences efficiently.  
This tool helps in validating database records, generating comparison reports, and improving data verification processes.

---

## Features

- Compare source and target datasets
- Detect missing and mismatched records
- Generate CSV comparison reports
- User-friendly frontend interface
- Backend processing using Java and Spring Boot
- Fast and efficient validation workflow

---

## Technologies Used

### Backend
- Java
- Spring Boot
- Maven
- REST API

### Frontend
- React.js
- Vite
- JavaScript
- HTML/CSS

### Database & Files
- SQL Database
- CSV Reports
